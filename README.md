# MvpArchitecture

### 网络层：

HTTP : Rxjava + Retrofit2 + Okhttp

ImageLoader:Glide

Cache: RxCache

### 数据处理层：

JavaBean: lombok

Java8：Stream + Lambda

Dependencies Injection: Dagger2

### UI层：

Views Binding: ButterKnife

Layout:ConstraintLayout

### 测试层：

AndroidTest: Espresso

UnitTest: Mockito + Robolectric
